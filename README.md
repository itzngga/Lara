# Lara
Whatsapp Bot Implementation of [Roxy](https://github/itzngga/Roxy)

## Requirements
- ffmpeg binary
- imagemagick binary
- libwebp binary
- tesseract ocr binary (indonesian lang included)

## How To
### setup .env
```shell
SQLITE_FILE=LARA.DB
```
### Run the project
```go
go run main.go
```
